Repozytorium dokumentuje moją naukę w zakresie projektowania relacyjnych baz danych oraz analizy danych biznesowych.
Silnik: Microsoft SQL Server
Środowiska: SQL Server Management Studio (SSMS) oraz Visual Studio Code (VS Code/Jupyter Notebooks)
W projektach używam T-SQL (Transact-SQL). W przeciwieństwie do standardowego SQL (który służy tylko do zapytań), T-SQL pozwala mi używać logiki programistycznej: zmiennych, procedur składowanych czy instrukcji warunkowych bezpośrednio w silniku bazy danych.

This repository documents my learning process in relational database design and business data analysis.
Engine: Microsoft SQL Server
Environments: SQL Server Management Studio (SSMS) and Visual Studio Code (VS Code / Jupyter Notebooks)
In my projects, I use T-SQL (Transact-SQL). Unlike standard SQL (which is used strictly for querying), T-SQL allows me to apply programming logic—such as variables, stored procedures, and conditional statements—directly within the database engine.
